WELCOME
=======

How to set me up under a fresh env

Requirements
---------
1. pip
2. virtualenvwrapper
3. postgresql running

Setup
---------

	$ mkvirtualenv text_your_city
	$ cdvirtualenv
	$ git clone git@github.com:yuletide/text_your_city.git
	$ pip install -r text_your_city/requirements.txt
